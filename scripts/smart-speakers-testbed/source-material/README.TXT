This directory contains all the source material for the voice assist experiments.

Source material are organized into .avi/.mov/.mp4/.mkv files, with matching name
subtitles. Any other file inside the directory is ignored.

Files in the source material directory will be processed in alphabetical order.
New files can be added at run-time.

NOTE: File "[S3.Ep10] Friday Night Tykes - Babies from Texas.mp4" contains
      just a black video and silent audio because of copyright laws, however
      the subtitles file has been preserved.
