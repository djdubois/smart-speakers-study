This directory contains all the captured material for the voice assist experiments.

Each capture has its own directory, with the same name of the source material.
Each directory contains the following files:

capture.mkv (raw video and audio from the camera)
capture.pcap (raw network traffic capture from tcpdump)
time.txt (contains start and end time of the capture in YYYY-XX-MM HH:MM:SS.XXX format)
